## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
# install dependencies
$ yarn install

# build project
$ yarn build
```

## Prisma Setup

```bash
# development
$ npx prisma generate

# watch mode
$ npx prisma dev
```

## Run tests

```
yarn gen-admin -e <email> -p <password>
```
